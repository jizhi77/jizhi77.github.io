webpack 配置

因为 @vue/cli-service 对 webpack 配置进行了抽象，如果想修改或者添加部分 webpack 的配置，可通过以下几种方式：

  1. @vue/cli-service 暴露出的可配置内容
    1. package.json 中的 vue  字段；
    2. vue.config.js 文件；
  2. @vue/cli-service 未暴露出的其他 webpack 配置内容
    1. vue.config.js  中的 configureWebpack 字段；
    2. vue.config.js 中的 chainWebpack 字段；

三、chainWebpack


Vue CLI 内部的 webpack 配置是通过 webpack-chain 维护的， vue.config.js 中也开放了针对此库的形式的配置。

总结：有些 webpack 选项是基于 vue.config.js 中的值设置的，所以不能修改。保险起见，将所有的 webpack配置项分为两类：

  1. vue cli 配置文件开放出来的配置项，也就是 vue.config.js 中支持的配置项，均在 vue.config.js 中配置；
  2. 非vue cli 开放出来的配置项均可以通过 configureWebpack 或者 chainWebpack 配置
